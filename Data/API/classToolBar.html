void ToolBar::SetLayout ( menu_layout  justification)

SetLayout supports three modes B_ITEMS_IN_ROW ,B_ITEMS_IN_COLUMN ,B_ITEMS_IN_MATRIX it tells the Toolbar how to layout the added Items.

so B_ITEMS_IN_ROW will place all items in a row, B_ITEMS_IN_COLUMN will place all items ... of course in a Column and B_ITEMS_IN_MATRIX will try to place all imtems like on a check board so that all items fit best.
