bool BColumn::AcceptsField ( const BField *  field) const [virtual]
Parameters:
fieldThe BField derivative to validate.

Implement this function on your BColumn derivatives to validate BField derivatives that your BColumn will be drawing/manipulating.

This function will be called when BFields are added to the Column, use dynamic_cast<> to determine if it is of a kind that your BColumn know how ot handle. return false if it is not.

Note:
The debugger will be called if you return false from here with information about what type of BField and BColumn and the logical field index where it occured.
Do not call the inherited version of this, it just returns true;
